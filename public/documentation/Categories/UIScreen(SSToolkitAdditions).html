Overview

Provides extensions to UIScreen for various common tasks.

Tasks

Accessing the Bounds

  • – currentBounds

    Returns the bounds of the screen for the current device orientation.
  • – boundsForOrientation:

    Returns the bounds of the screen for a given device orientation. UIScreen’s bounds method always returns the bounds of the screen of it in the portrait orientation.

Screen Attributes

  • – isRetinaDisplay

    Returns a Boolean indicating if the screen is a Retina display.

Instance Methods

boundsForOrientation:

Returns the bounds of the screen for a given device orientation. UIScreen’s bounds method always returns the bounds of the screen of it in the portrait orientation.

- (CGRect)boundsForOrientation:(UIInterfaceOrientation)orientation

Parameters

orientation

The orientation to get the screen’s bounds.

Return Value

A rect indicating the bounds of the screen.
