SSRatingPicker Class Reference

Inherits from UIControl
Declared in SSRatingPicker.h

Overview

A simple control for picking a rating.

The default configuration matches the rating picker in the App Store app (to the pixel). The textLabel is automatically faded in if there is no rating and faded out when there is a rating.

To receive changes when the selectedNumberOfStars changes, add a target for the UIControlEventValueChanged control event (similar to UISegmentedControl).
